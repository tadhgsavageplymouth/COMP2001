
This is a simple Profile Service App that provides APIs for managing user profiles, followers, followings, and more. It's built using ASP.NET Core and Entity Framework, and it uses AutoMapper for object mapping.

## Getting Started

Follow these steps to get the app up and running on your local machine.

### Prerequisites

- [.NET 6 SDK](https://dotnet.microsoft.com/download/dotnet/6.0)
- [Visual Studio Code](https://code.visualstudio.com/) (recommended) or another code editor of your choice
- [GitHub](https://github.com/) account (optional, if you want to clone the repository)
